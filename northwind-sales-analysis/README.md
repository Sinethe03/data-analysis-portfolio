# Northwind Sales Operations Analysis

## Overview
This project analyses sales operations using the Northwind sample database.
The objective is to demonstrate SQL skills for working with **relational databases**, including joins, aggregations, and business-focused analysis.

The analysis focuses on understanding sales performance across products, customers, employees, and time.

---

## Business Questions Answered
- How does total sales revenue trend over time?
- Which products generate the most revenue?
- Who are the top customers by revenue?
- How do individual employees perform in terms of sales?
- What is the average order value?

---

## Tools & Technologies
- SQL (SQLite)
- DB Browser for SQLite
- Excel (summary reporting)

---

## Dataset
The Northwind database is a classic sample business dataset containing:
- Orders
- Order Details
- Products
- Customers
- Employees
- Categories

The dataset represents a fictional trading company and is commonly used to demonstrate relational database concepts.

---

## Analysis Approach
- SQL joins were used to combine data across multiple related tables
- Revenue was calculated using unit price, quantity, and discounts
- Aggregations and grouping were applied to answer business-level questions
- Results were structured for easy export into Excel for reporting
